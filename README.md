# 🌡️ IoT-based Temperature & Humidity Monitoring System

본 프로젝트는 **DHT11 온·습도 센서**를 사용하여 주변 환경의 온도와 습도를 측정하고,
특정 조건 발생 시 **카카오톡 메시지 알림**을 전송하는 **팀 프로젝트 기반 환경 모니터링 시스템**입니다.
Python을 활용하여 센서 데이터 수집 및 외부 API 연동을 구현하였습니다.

---

## 📌 Project Overview

* DHT11 센서를 이용한 온·습도 데이터 측정
* Python 기반 센서 데이터 처리
* 카카오톡 API를 활용한 실시간 알림 전송
* IoT 및 센서 기반 시스템 설계 경험

---

## 🛠️ Tech Stack

* **Hardware**: DHT11 Temperature & Humidity Sensor
* **Platform**: Raspberry Pi (또는 Linux 기반 환경)
* **Language**: Python
* **API**: KakaoTalk Message API

---

## ⚙️ Main Features

* 🌡️ 실시간 온도 측정
* 💧 실시간 습도 측정
* ⏱️ 주기적인 센서 데이터 수집
* 💬 임계값 초과 시 카카오톡 알림 전송

---

## 🔔 KakaoTalk Notification

* 카카오톡 API 토큰을 이용한 메시지 전송
* 온도 또는 습도가 설정된 기준값을 초과할 경우 자동 알림
* 원격 환경 모니터링 가능
